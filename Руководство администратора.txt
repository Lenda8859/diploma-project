Руководство администратора
Введение
Администраторы системы отвечают за настройку, поддержку и мониторинг работы приложения. 
Они могут управлять данными системы, следить за логами, обновлять программное обеспечение и проводить миграции базы данных.

Установка и настройка:
Создание виртуального окружения:
Выполните команду: python -m venv venv.
Активируйте окружение:
Windows: venv\Scripts\activate
Linux/macOS: source venv/bin/activate

Установка зависимостей:
Установите все необходимые зависимости командой: pip install -r requirements.txt.

Настройка базы данных:
В системе используется база данных SQLite (hotel_management.db), которая уже настроена и готова к использованию.
Миграции можно выполнить командой: python manage.py migrate (для веб-приложения).

Логирование и мониторинг:
Все логи работы системы сохраняются в папке logs. Рекомендуется регулярно проверять логи для отслеживания ошибок и других событий системы.

Обновление системы:
Для обновления системы следует загрузить новые версии файлов и провести миграции, если это необходимо.
